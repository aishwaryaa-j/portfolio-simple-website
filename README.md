# 🌸 Simple Portfolio Design

Welcome to my little portfolio project! ✨  
This is a clean and minimal portfolio design created with simplicity and elegance in mind.

---

## 💫 About the Design

This portfolio includes just two pages:

- 🏠 **Home** – A simple and welcoming introduction  
- 📬 **Contact** – An easy way to connect  

The goal of this design is to keep things neat, user-friendly, and visually pleasing without any unnecessary clutter.

---

## 🎨 Design Style

- 🌿 Soft and minimal layout  
- ✍️ Clean typography  
- 📐 Balanced spacing for better readability  

---

## 🚀 Features

- 🔗 Easy navigation between pages  
- ⚡ Lightweight and simple structure  
- 🌱 Beginner-friendly design  
- 💡 Focused on user experience  

---

## 💌 Why This Project?

Sometimes less is more!  
This project shows how a portfolio can still look beautiful and professional with just the essentials.
